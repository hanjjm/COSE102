## **Tab1(전화번호부)**
1. Update 버튼을 누르면 현재 핸드폰에 저장되어 있는 전화번호부 목록을 불러옵니다(ListView).
2. 어플리케이션이 켜져 있는 도중에 전화번호부에 새로운 목록이 추가된다고 해도, updata 버튼을 누르면 바로 update가 됩니다.

## **Tab2(Gallery)**
1. Update 버튼을 누르면 현재 핸드폰에 저장되어 있는 앨범에 사진 목록을 불러옵니다(GridView).
2. 위와 동일하게 어플리케이션이 켜져 있는 도중에 사진을 찍어도, update 버튼을 누르면 바로 update가 됩니다.

## **Tab3(Favorite List)**
1. 전화번호부 목록에서 친한친구 등록을 한 list를 띄어줍니다(ListView).
2. 우측 하단에 하트 아이콘을 누르면 PlusFriend Activity로 이동합니다.
3. PlusFrined Activity는 전화번호부에서 친한친구 등록을 하지 않은 list를 띄어줍니다(ListView).
4. 각 item들을 클릭하면 우측에 star imageview의 색이 gray -> yellow로 바뀌고, 이 목록들은 오른쪽 하단에 하트 아이콘을 누르면 favorite list에 등록이 됩니다.
5. Tab3에 다시 간 뒤, update 버튼을 누르면 Plus Activity에서 favorite list로 등록해놓은 친구들 list가 생성됩니다.
6. Favorite List에 등록된 친구 item을 다시 클릭을 하면 친한 친구 등록이 해제되어, 다시 PlusFriend Activity로 클릭한 item이 이동합니다.
7. 이 item들이 이동하는 일련의 과정들은 모두 내장된 Database에 저장이 되므로, 어플리케이션을 종료하고 다시 실행시킨다고 해도 list는 그대로 남아있습니다.
8. 하단에 카메라 버튼을 클릭하면, 현재 띄어져 있는 목록이 캡쳐가 되어 카메라 앨범으로 자동으로 들어갑니다.
